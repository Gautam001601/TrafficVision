Training Dynamics

Trained with Ultralytics YOLOv8 for 25-30 epochs

Loss Convergence: Box Loss ↓ 1.5 → 1.15, Class Loss ↓ 2.5 → 0.9. Validation stabilized around epoch 20.

Accuracy Growth: mAP@0.5 climbed from 0.15 to 0.383. Recall reached ~0.55.

🛠️ Technical Stack

Model: YOLOv8 Nano (3.2M)
Framework: PyTorch / Ultralytics
Data: Roboflow
Augmentation: Mosaic, Flip, Rotate
